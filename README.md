project  implements aclient/server medical platform using TCP API in Java.
The platform allows patients to submit tickets that include medical questions or
consultations. The platform has registered doctors that are responsible for responding to
these tickets.
The server application must accept connections from two types of users. The first type
represents doctors whose connection requires logging in using a username and password.
The other type represents patients, whose connection does not require them to log-in to
submit tickets.
